# Student Management System

##  About

A full-stack web application for managing student records.

I built this project to practice **Java Spring Boot** and **React**.

## Tech Stack 

* **Backend:** Java 21, Spring Boot, Spring Data JPA

* **Frontend:** React, TypeScript

* **Database:** MySQL

* **Tools:** VS Code

## Features 

* **View Students:** Display a list of all students.

* **Add Student:** Create new student records.

* **Update Student:** Edit student details (Name, Email, GPA).

* **Delete Student:** Remove student records.

* **Data Validation:** Ensure GPA is between 0.0 and 4.0.

## How to Run(Two Terminals)

### 1. Backend (Java) 

Open a terminal and run:

cd backed

mvnw spring-boot:run

### 2. Frontend (React):

Open a new terminal and run:

cd frontend

pnpm install

pnpm dev
